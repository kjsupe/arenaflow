# Support

ArenaFlow is free software released for general use by FEC operators.

There is no guaranteed support, no 24/7 support, and no emergency operational support. If you run ArenaFlow at your business, you are responsible for testing it, backing it up, and deciding whether it fits your operation.

## Before Asking For Help

Please read:

- [README.md](README.md)
- [docs/INSTALL.md](docs/INSTALL.md)
- [docs/USER_GUIDE.md](docs/USER_GUIDE.md)
- [SECURITY.md](SECURITY.md)

## GitHub Issues

Use GitHub Issues for:

- Bug reports.
- Documentation corrections.
- Feature requests.

When reporting a bug, include:

- ArenaFlow version or commit.
- Docker or local Python install.
- Operating system.
- Browser.
- What you expected.
- What happened instead.
- Relevant logs with private data removed.

Do not post:

- `.env` contents.
- Real customer names.
- Ticket codes or PINs.
- Public server admin URLs you do not want shared.
- Database backups.
